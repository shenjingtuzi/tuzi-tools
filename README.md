# tuzi-tools - 多功能文档处理工具服务

tuzi-tools 是一个基于 Spring Boot 开发的多功能文档处理工具服务，专注于提供高效、便捷的办公文档格式转换与压缩功能。项目集成了 Aspose 系列库，支持 Excel、Word、PDF 等多种文档的处理，可作为后端服务快速对接各类前端应用。

## 核心功能

### Excel 处理

- **格式转换**：支持将 .xlsx、.xls 格式文件转换为多种目标格式（如转换为图片时会生成包含所有页的 ZIP 包）
- **文件压缩**：提供不同级别的 Excel 文件压缩功能，减小文件体积的同时保持内容完整性

### Word 处理

- 支持 Word 文档格式转换与压缩（基于 Aspose.Words 实现）

### PDF 处理

- 提供 PDF 水印添加（支持图片水印等）等功能（基于 Aspose.PDF 实现）

### 通用特性

- **跨域支持**：内置跨域配置，方便小程序、Web 等前端应用调用
- **文件管理**：自动处理上传文件的存储与临时文件清理
- **安全控制**：集成 JWT 工具类，支持接口访问权限控制

## 技术栈

- **框架**：Spring Boot
- **核心依赖**：Aspose 系列库（Aspose.Cells、Aspose.Words、Aspose.PDF 等）
